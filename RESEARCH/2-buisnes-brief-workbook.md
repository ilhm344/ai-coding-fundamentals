# 📋 Бизнес-бриф: исследование рынка за 60 минут

> **Задача:** провести исследование рынка и заполнить бизнес-бриф для воркшопа.
> В конце у тебя будет готовый файл, который мы загрузим в AI для создания лендинга и бота.
>
> **Как устроен документ:** на каждом шаге есть готовый промпт — скопировал, вставил свои данные, получил результат. AI делает большую часть работы, ты — финальную проверку.

**Что нужно:** браузер, Claude или ChatGPT, 60 минут.

> 💡 **Как работать с этими файлами:**
> Этот файл — инструкции и промпты. Заполняй **бизнес-бриф-ШАБЛОН.md** — он без лишнего текста, только поля для ввода.

---

## ШАГ 1 — AI анализирует конкурентов (20 минут)

### 1.1 Найди конкурентов (5 мин, руками)

Открой Яндекс, вбей 2–3 запроса — как тебя ищут клиенты.
Например: *«веб-студия Екатеринбург»*, *«сделать лендинг для бизнеса»*.

Собери 5 ссылок на сайты конкурентов из выдачи и запиши их в шаблон.

> 💡 Бери первые 5 из органики (не реклама). Это главные конкуренты по поисковому трафику.

---

### 1.2 Отдай ссылки AI (15 мин, промпт)

Скопируй промпт ниже, вставь свои ссылки и отправь в Claude или ChatGPT.
Результат вставь в шаблон в раздел «ШАГ 1».

```
Ты — маркетинговый аналитик. Я собираю данные о конкурентах перед
созданием лендинга для своего бизнеса.

МОЙ БИЗНЕС: [напиши в 1-2 предложениях чем занимаешься]
МОЯ НИША: [например: веб-студия, барбершоп, юрист]
МОЙ ГОРОД: [если локальный бизнес]

САЙТЫ КОНКУРЕНТОВ:
1. [ссылка]
2. [ссылка]
3. [ссылка]
4. [ссылка]
5. [ссылка]

Зайди на каждый сайт и заполни таблицу:

| Параметр | Конк. 1 | Конк. 2 | Конк. 3 | Конк. 4 | Конк. 5 |
|---|---|---|---|---|---|
| Заголовок первого экрана | | | | | |
| УТП / главный оффер | | | | | |
| Цены (если указаны) | | | | | |
| Что на кнопке CTA | | | | | |
| Соцдоказательства (отзывы/кейсы) | | | | | |
| Позиционирование (бюджет/средний/премиум) | | | | | |
| Главная слабость / что раздражает | | | | | |

После таблицы дай выводы:
1. Что пишут ВСЕ (клише, которые надо избегать)
2. Где у всех дыра — чего никто не делает, но клиенты хотят
3. Ценовой диапазон рынка — от и до
4. Какие 2-3 точки дифференциации открыты для нового игрока
```

## ШАГ 2 — Читаешь отзывы сам (15 минут)

Этот шаг — только руками. AI не знает, что пишут о конкретных компаниях в твоём городе прямо сейчас.

**Где смотреть:**
- **Яндекс Карты** — найди 2–3 конкурента, открой отзывы
- **2ГИС** — особенно если ты в регионе
- **VC.ru** — для диджитал-ниш (агентства, фрилансеры)

> 💡 Читай **1-звёздочные и 5-звёздочные** отзывы. Средние — вода. Крайности — золото.
> Тебе нужно прочитать 20–30 отзывов суммарно по 2–3 конкурентам.

Выпиши цитаты дословно в шаблон — как написал человек.

---

### 2.1 AI находит паттерны в отзывах (промпт)

Собрал цитаты — теперь скорми их AI.
Результат вставь в шаблон в раздел «ШАГ 2».

```
Вот отзывы клиентов конкурентов в нише [твоя ниша].

НЕГАТИВНЫЕ ОТЗЫВЫ:
[вставь цитаты]

ПОЗИТИВНЫЕ ОТЗЫВЫ:
[вставь цитаты]

Проанализируй и дай мне:

1. ТОП-5 болей клиентов — что их бесит больше всего
   Формат: боль → как клиент это формулирует своими словами

2. ТОП-5 вещей, которые клиенты ценят больше всего

3. Самые частые возражения перед покупкой
   (что останавливает от выбора исполнителя)

4. Готовые фразы для лендинга — на языке клиента, не маркетинговом.
   Например: не "высокое качество работ", а "сделали именно так,
   как я хотел — с первого раза"

5. Какую боль закрыть в УТП, чтобы выделиться на этом рынке
```

## ШАГ 3 — Про твой бизнес (15 минут, руками)

Здесь AI тебе не поможет — только ты знаешь правду о своём бизнесе.
Пиши честно и просто, без маркетингового лоска. AI потом упакует.

Заполни раздел «ШАГ 3» в шаблоне.

---

## ШАГ 4 — AI формулирует УТП (промпт)

Теперь собери всё из шаблона и отдай AI — пусть сформулирует твоё позиционирование.
Результат вставь в шаблон в раздел «ШАГ 4».

```
Помоги сформулировать УТП для моего бизнеса.

МОЙ БИЗНЕС:
[вставь описание из шага 3]

УСЛУГИ И ЦЕНЫ:
[вставь из шага 3]

ЦЕЛЕВАЯ АУДИТОРИЯ:
[вставь из шага 3]

МОИ СИЛЬНЫЕ СТОРОНЫ:
[вставь из шага 3]

МОИ СЛАБОСТИ:
[вставь из шага 3]

КОНКУРЕНТЫ И ИХ ПОЗИЦИОНИРОВАНИЕ:
[вставь таблицу из шага 1]

КЛИШЕ РЫНКА (чего надо избежать):
[вставь из шага 1]

ДЫРЫ РЫНКА (возможности):
[вставь из шага 1]

ГЛАВНЫЕ БОЛИ КЛИЕНТОВ:
[вставь из шага 2]

ЯЗЫК КЛИЕНТОВ (цитаты из отзывов):
[вставь из шага 2]

Дай мне:

1. ГЛАВНОЕ УТП — одно предложение, которое объясняет чем я
   отличаюсь. Без клише. Конкретно.

2. ЗАГОЛОВОК для первого экрана лендинга — цепляющий,
   на языке клиента, закрывает главную боль.

3. ПОДЗАГОЛОВОК — 1-2 предложения, раскрывают заголовок.

4. 3 ВАРИАНТА ПОЗИЦИОНИРОВАНИЯ в разных стилях:
   — Рациональный (измеримая выгода, цифры)
   — Эмоциональный (как клиент себя чувствует)
   — Через проблему (боль → решение)

5. ТОП-5 ВОЗРАЖЕНИЙ клиентов и ответы на них —
   для FAQ-блока лендинга и базы знаний бота.
```

## ШАГ 5 — Финальная проверка (5 минут)

Открой шаблон и ответь на три контрольных вопроса в разделе «ШАГ 5».
Если не можешь ответить чётко — вернись и дозаполни нужный шаг.

## ✅ Готово

Шаблон заполнен. На воркшопе мы загрузим его в AI и он сделает:
- структуру продающего лендинга
- тексты и заголовки
- базу знаний для Telegram-бота

**Совет:** не шлифуй формулировки — пиши как есть. AI разберётся. Главное — честные данные и реальные цитаты из отзывов.
